# F.softmax(inputs, dim=1) 结果详解

## 输入数据

```python
import torch
import torch.nn.functional as F

# inputs: shape (2, 4, 4) = (batch_size, num_classes, sequence_length)
inputs = torch.tensor([
    # 样本 0: "John lives in Paris"
    [
        [0.1, 0.2, 0.3, 0.4],  # 位置 0 (John): 4 个类别的 logits
        [2.0, 0.5, 0.3, 0.2],  # 位置 1 (lives): 4 个类别的 logits
        [0.3, 0.1, 0.2, 0.4],  # 位置 2 (in): 4 个类别的 logits
        [0.2, 0.1, 0.1, 2.5]   # 位置 3 (Paris): 4 个类别的 logits
    ],
    # 样本 1: 另一个句子
    [
        [1.5, 0.5, 0.3, 0.2],
        [0.2, 0.3, 0.4, 0.1],
        [0.1, 0.2, 0.3, 0.4],
        [0.3, 0.4, 0.2, 0.1]
    ]
], dtype=torch.float32)

print(f"inputs shape: {inputs.shape}")  # torch.Size([2, 4, 4])
```

## Softmax 计算

```python
probs = F.softmax(inputs, dim=1)
# dim=1 表示在类别维度（第 2 维）上计算 softmax
# 对于每个位置，将 4 个类别的 logits 转换为概率分布
```

## 计算过程详解

### 样本 0 的计算

#### 位置 0 (John) - 4 个类别的 logits: [0.1, 2.0, 0.3, 0.2]

```
步骤 1: 找最大值
  max = 2.0

步骤 2: 减去最大值（数值稳定性）
  shifted = [0.1-2.0, 2.0-2.0, 0.3-2.0, 0.2-2.0]
          = [-1.9, 0.0, -1.7, -1.8]

步骤 3: 计算 exp
  exp_shifted = [exp(-1.9), exp(0.0), exp(-1.7), exp(-1.8)]
              = [0.150, 1.000, 0.183, 0.165]

步骤 4: 求和
  sum = 0.150 + 1.000 + 0.183 + 0.165 = 1.498

步骤 5: 归一化
  probs[0, :, 0] = [0.150/1.498, 1.000/1.498, 0.183/1.498, 0.165/1.498]
                  = [0.100, 0.668, 0.122, 0.110]
```

#### 位置 1 (lives) - 4 个类别的 logits: [0.2, 0.5, 0.3, 0.2]

```
步骤 1: max = 0.5
步骤 2: shifted = [-0.3, 0.0, -0.2, -0.3]
步骤 3: exp_shifted = [0.741, 1.000, 0.819, 0.741]
步骤 4: sum = 3.301
步骤 5: probs[0, :, 1] = [0.225, 0.303, 0.248, 0.225]
```

#### 位置 2 (in) - 4 个类别的 logits: [0.3, 0.1, 0.2, 0.4]

```
步骤 1: max = 0.4
步骤 2: shifted = [-0.1, -0.3, -0.2, 0.0]
步骤 3: exp_shifted = [0.905, 0.741, 0.819, 1.000]
步骤 4: sum = 3.465
步骤 5: probs[0, :, 2] = [0.261, 0.214, 0.236, 0.289]
```

#### 位置 3 (Paris) - 4 个类别的 logits: [0.2, 0.1, 0.1, 2.5]

```
步骤 1: max = 2.5
步骤 2: shifted = [-2.3, -2.4, -2.4, 0.0]
步骤 3: exp_shifted = [0.100, 0.091, 0.091, 1.000]
步骤 4: sum = 1.282
步骤 5: probs[0, :, 3] = [0.078, 0.071, 0.071, 0.780]
```

## 完整代码示例

```python
import torch
import torch.nn.functional as F

# 输入 logits
inputs = torch.tensor([
    [
        [0.1, 0.2, 0.3, 0.4],
        [2.0, 0.5, 0.3, 0.2],
        [0.3, 0.1, 0.2, 0.4],
        [0.2, 0.1, 0.1, 2.5]
    ],
    [
        [1.5, 0.5, 0.3, 0.2],
        [0.2, 0.3, 0.4, 0.1],
        [0.1, 0.2, 0.3, 0.4],
        [0.3, 0.4, 0.2, 0.1]
    ]
], dtype=torch.float32)

# 计算 softmax
probs = F.softmax(inputs, dim=1)

print("Softmax 结果:")
print(probs)
print(f"\n形状: {probs.shape}")

# 验证每个位置的概率和为 1
print("\n每个位置的概率和:")
print(probs.sum(dim=1))  # 应该都是 1.0

# 查看样本 0，位置 0 的概率分布
print("\n样本 0，位置 0 的概率分布:")
print(probs[0, :, 0])
print(f"总和: {probs[0, :, 0].sum().item():.6f}")  # 应该是 1.0
```

## 总结

| 特性 | 说明 |
|------|------|
| **输入** | `inputs` shape: `(N, C, L)`，包含 logits |
| **操作** | `F.softmax(inputs, dim=1)` 在类别维度上归一化 |
| **输出** | `probs` shape: `(N, C, L)`，包含概率 |
| **数值范围** | `[0, 1]` |
| **约束** | 每个位置的所有类别概率之和 = 1 |
| **含义** | `probs[i, j, k]` = 样本 i，类别 j，位置 k 的概率 |

**关键理解**:
- `dim=1` 表示在类别维度上计算 softmax
- 对于每个位置，将 4 个类别的 logits 转换为概率分布
- 每个位置的所有类别概率之和为 1
- 数值越大，模型越倾向于该类别
